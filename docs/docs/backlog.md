# Product Backlog — Hospital / Clinic Management System

## User Stories

| ID | User Story | Priority | Story Points | Acceptance Criteria |
|----|-----------|----------|--------------|-------------------|
| US-01 | As a patient, I want to register my personal information so that the clinic has my records. | High | 3 | Patient can fill out and submit a registration form with name, age, address, and contact number. |
| US-02 | As a patient, I want to book an appointment online so that I can schedule a visit without calling. | High | 5 | Patient can select a doctor, date, and time slot and receive a confirmation. |
| US-03 | As a doctor, I want to view my daily schedule so that I know which patients I will see. | High | 3 | Doctor can log in and see a list of appointments for the current day. |
| US-04 | As a receptionist, I want to manage patient check-ins so that the clinic flow runs smoothly. | High | 3 | Receptionist can mark a patient as arrived and update their status in the system. |
| US-05 | As a doctor, I want to record a patient's diagnosis and prescription so that medical history is saved. | High | 5 | Doctor can enter diagnosis notes and prescriptions that are saved to the patient's profile. |
| US-06 | As an admin, I want to manage doctor profiles so that clinic information stays up to date. | Medium | 3 | Admin can add, edit, and deactivate doctor accounts. |
| US-07 | As a patient, I want to view my medical history so that I can track my past visits and treatments. | Medium | 5 | Patient can log in and view a list of past appointments, diagnoses, and prescriptions. |
| US-08 | As a receptionist, I want to generate patient billing so that payments are properly tracked. | Medium | 5 | Receptionist can create a bill based on services rendered and mark it as paid or unpaid. |
| US-09 | As an admin, I want to generate reports on clinic operations so that I can monitor performance. | Low | 8 | Admin can view monthly reports showing number of patients, appointments, and revenue. |
| US-10 | As a patient, I want to receive appointment reminders via email so that I don't miss my schedule. | Low | 5 | System sends an automated email reminder 24 hours before the appointment. |
